# Friday AI Platform Modules

Modules are product capabilities that use J.A.R.V.I.S Core contracts.

They are intentionally placeholders in this foundation sprint. No module implements
business behavior yet.

## Initial Modules

- Trading
- Finance
- Marketing
- SEO
- Documents
- Automation
- Sites
- CRM

## Boundary

Modules never talk directly to providers. Provider access must pass through Core
contracts and orchestration.
